# Next.js + Tailwind + TypeScript Template

This is a quick start template, using the latest versions of NextJS, Tailwind and TypeScript with some other useful development tools included.

## What's Included?

* ### NextJS
* ### TypeScript
* ### CSS
    * Tailwind
    * SASS
    * HeadlessUI
    * HeroIcons
* ### State Management
    * Zustand
* ### Forms
    * React Hook Form
* ### Testing
    * React Testing Library
    * Jest
    * Cypress
* ### Commit Tools
    * Commitlint
    * Commitizen
    * Husky
    * Lint Staged
* ### Util
    * Sort Package JSON
    * NPM Check Updates
    * Prettier
    * ESLint
